# Yutaka Morita Hub

An experimental personal hub page designed primarily as a destination from a business card QR code.

This project explores how a personal “hub” can connect:
- UX
- AI
- Human decision making
- Structure thinking
- Creative identity

Rather than building a traditional portfolio, the goal is to create a small digital space that feels more like an entrance to ongoing thoughts, projects, and experiments.

The site is intentionally lightweight:
- Single HTML page
- GitHub Pages hosting
- Minimal dependencies
- Long-term maintainability

It also serves as an experiment in networking, personal branding, and experience design.

“Perfect moments matter too.” 🍨
